# AWS 인프라

## EC2
- 가상 컴퓨터
- 리전 = vpc + s3

## iam
- 접근 권한 부여

## vpc(virtual private cloud)
- 가상 네트워크

## s3
- 스토리지
- 데이터 저장 및 정적인 웹 페이지 호스팅
- 버킷 내에 저장 (파일 + 메타데이터)
- 버전관리 가능

## 데이터베이스
- RDS = 행, 열로 접근 (SQL), 관계형, 관리부담을 최소화
- dynamoDB = 키로 접근 (noSQL), 완성형, 저렴한 비용

## 관리
- auto scaling = 사용량에 따라 ec2 인스턴스 갯수가 자동으로 변함, 확장성
- 로드 벨런서 = 확장성
- cloudwatch = 모니터링
